# Agentic Immunology

## Local Agent
navigate to the main folder of the repo.

**venv**
```bash
python3 -m venv iagent_env
source iagent_env/bin/activate
pip install -r requirements.txt
```

**conda**
```bash
conda create -p ./iagent_env python=3.11 -y
conda activate ./iagent_env
pip install -r requirements.txt
```

```bash
bash agent/run.sh
```
